# Chat 1-on-1

Chat 1-on-1 is a MERN Stack Chatting App.
Uses Socket.io for real time communication and stores user details in encrypted format in MongoDB Database.

## Tech Stack

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB
